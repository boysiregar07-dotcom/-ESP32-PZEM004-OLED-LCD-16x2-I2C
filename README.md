# Monitoring Daya Listrik ESP32 + PZEM004T + OLED

Proyek ini bertujuan untuk memonitor tegangan, arus, dan daya listrik menggunakan sensor **PZEM004T** dan **ESP32**, serta menampilkan hasilnya di **OLED Display**.

##  Komponen
- ESP32 DevKit V1
- Sensor PZEM004T
- OLED 16x2 I2C
- Kabel jumper

##  Wiring Diagram


## Fitur
- Pembacaan tegangan, arus, daya
- Tampilan real-time di OLED
- Komunikasi serial ke PC

##  Library yang Digunakan
- `PZEM004Tv30.h`
- `Adafruit_SSD1306.h`
- `Wire.h`
- `LiquidCrystal_I2C.h`
## Cara Penggunaan
1. Upload kode dari folder `src/` ke ESP32  
2. Hubungkan sensor dan OLED sesuai diagram  
3. Lihat hasil di OLED atau Serial Monitor
